# 快速开始

根据项目需要引入 `Open XML SDK` 库。

<!-- tabs:start -->

#### **.NET CLI**

```dotnet
dotnet add package DocumentFormat.OpenXml
```

#### **Visual Studio**

在 Visual Studio 中打开项目或解决方案，然后选择 “工具>NuGet 包管理器>包管理器控制台 ”以打开“包管理器控制台”窗口。

```powershell
Install-Package DocumentFormat.OpenXml
```
